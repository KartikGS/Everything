> This file contains Everything-specific file paths for reporting discrepancies. Read the general protocol first, then apply these overrides.

## Everything Reporting File Paths

### Sub-Agent → Tech Lead
Document the blocker in the role's report file:
- `/projects/Everything/context/agents/conversations/<role>-to-tech-lead.md`
- Section: `## BLOCKER / FEEDBACK`

### Tech Lead → BA Agent
- File: `/projects/Everything/context/agents/conversations/tech-lead-to-ba.md`
- Section: `## REQUIREMENT FEEDBACK`

## Reference to General Protocol

All discrepancy types (Missing Contract, False Assumption, Logical Flaw), the resolution protocol, priority rules, and environmental blocker validation steps are defined in the general protocol linked above.
