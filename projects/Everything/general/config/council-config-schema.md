# Council Config Schema

> `council-config.json` is generated by `scripts/init.js` and placed at the root of a project directory: `projects/<project-name>/council-config.json`.
>
> It is project-specific — it never lives inside `general/`.

---

## Full Schema

```json
{
  "project": "string — project name, matches the directory name under projects/",
  "created": "string — ISO date (YYYY-MM-DD) of when the council was configured",
  "council": {
    "normal_pos": [
      {
        "id": "string — stable identifier, format: po-N (e.g. po-1, po-2)",
        "name": "string — human-readable name given to this council member",
        "perspective": "string — lens this member brings (e.g. user-focused, business-focused)"
      }
    ],
    "devils_advocates": [
      {
        "id": "string — stable identifier, format: da-N (e.g. da-1)",
        "name": "string — human-readable name given to this DA member",
        "perspective": "string — lens this member brings (e.g. risk-focused)"
      }
    ]
  }
}
```

---

## Constraints

| Field | Rule |
|-------|------|
| `normal_pos` length | 2, 3, or 4 members |
| `devils_advocates` length | `floor(normal_pos.length / 2)` — always 1 or 2 |
| `id` values | Must be unique within their array; stable across sessions |
| `name` | Free text; used as the display name in forum summaries |
| `perspective` | Free text; loaded into each agent's context to shape their contribution |

## Example (2 Normal POs, 1 DA)

```json
{
  "project": "MyProject",
  "created": "2026-02-22",
  "council": {
    "normal_pos": [
      { "id": "po-1", "name": "Alex", "perspective": "user-focused" },
      { "id": "po-2", "name": "Jordan", "perspective": "business-focused" }
    ],
    "devils_advocates": [
      { "id": "da-1", "name": "Morgan", "perspective": "risk-focused" }
    ]
  }
}
```


---

## No-Council Projects

If the user selected "No council" during `init.js`, no `council-config.json` is created. The project's `context/AGENTS.md` will reflect this and direct deliberation to standard BA/Tech Lead roles.
